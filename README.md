# ComfyUI Stack

A production-ready Docker image for running [ComfyUI](https://github.com/comfyanonymous/ComfyUI) with NVIDIA CUDA support.

## Features

- GPU-accelerated (Nvidia CUDA only)
- Flash Attention
- Sage Attention
- Pre-configured environment for ComfyUI
- Do not run as root
- Install / Uninstall extensions
